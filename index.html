Projects

Automatic Projectile Refiller

What?

  • Able to load projectile in the magazine loader through a hopper system connected to a funnel
  • The Hopper system allows only one projectile to go through at a time, which prevents any blockage

How?

  • Took the circuit board from a previously designed paintball hopper, which was designed to vibrate and prevent blockage
  • 3D printed a board where I attached the circuit board
  • Attach the board to a funnel where all the projectiles will be placed

Results

  • Removed the need for the worker to manually fill the magazine loader
  • Allows testing staff to load magazines with ease and focus more time on testing and validating

Custom Arbor Press

What?

  • Able to load projectiles into the magazine with assistance from a handle
  • Enables exactly 5 projectiles to come in at a time, allowing factory workers to fill every magazine with the correct number of projectiles

How?

  • 3D-designed components in Creo Parametrics
  • Used wood to shape the base of the arbor press and used a spring, which allows the handle to go back to its neutral position after every use

Results

  • Decrease the time to load the magazine by 80%
  • Allows the worker to spend less time loading, which therefore allows more testing and leads to more shipping of products

Speed Magazine Loader

What?

  • A box with 2 nose plungers on both sides, which allows only 5 projectiles to go in at a time
  • A plunger with a flat face allows an equal force in the projectile to allow them to be loaded in a magazine

How?

Results

  • Increase the productivity of the testing staff
  • Allows more attention to testing and validating

O-ring install Mandrel tool

What?

  • Able to assist the factory workers in installing an O-ring in the non-lethal airgun
  • Shoulders in the tool allows O-ring to rest while being installed in parts

How?

  • Back-engineered a current production through measurement from Keyence Inspection
  • Used Creo Parametrics to 3D design the mandrel

Results

  • Increased efficiency in factory workers through the ease of use
  • Manufactured from stainless steel, allowing the part to be sturdy while keeping the cost low

Hammer Spring Install Tool

What?

  • A spring install tool that is manufactured with aluminum and a rubberized grip handle
  • Rubberized handle prevents any slip when installing the hammer spring in the product

How?

  • Designed the base in Creo Parametric to hold two pin gauges
  • Press-fitting the gauge pin in the hole allows a tighter fit
  • Used Loctite in the threaded handle, causing the handle to be very sturdy

Results

  • Allows the worker to install the hammer spring at a much faster rate
  • Able to shut off the machine quickly during emergencies, preventing deadly injuries to factory workers

Electrical Machine Guard

What?

  • Protects workers with a switch interface that allows them to turn it off with a flip of a switch

How?

  • Soldered switches through Arduino-coded shutdown

Results

  • Increased factory safety by 43%
  • Lower cost implementation
